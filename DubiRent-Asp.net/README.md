# DUBiRent - Luxury Stays in Dubai

Модерна уеб страница за наемане на луксозни имоти в Дубай.

## Структура на проекта

- `index.html` - Главна страница с всички секции
- `script.js` - JavaScript за интерактивност
- `README.md` - Документация

## Технологии

- **HTML5** - Структура на страницата
- **Tailwind CSS** - Стилизиране (чрез CDN)
- **JavaScript** - Интерактивност и функционалност
- **Font Awesome** - Икони

## Секции на страницата

1. **Header** - Навигация с лого и меню
2. **Hero Section** - Главна секция с търсене и галерия
3. **Luxury Properties** - Карти с имоти
4. **Why Choose DubiRent** - 6 карти с предимства и статистики
5. **Popular Locations** - Популярни локации в Дубай
6. **How It Works** - 3 стъпки как работи услугата
7. **Find Your Perfect Long-Term Home** - Информация за дългосрочни наеми
8. **Footer** - Долен колонтитул

## Как да стартирате

1. Отворете `index.html` в браузър
2. Или стартирайте локален сървър:
   ```bash
   # С Python
   python -m http.server 8000
   
   # С Node.js (http-server)
   npx http-server
   ```

## Цветова схема

- **Червен (Red)**: `#DC2626` - Основен акцентен цвят
- **Бял (White)**: `#FFFFFF` - Фон
- **Сив (Gray)**: Различни нюанси за текст и фонове

## Функционалности

- Smooth scrolling навигация
- Търсене на имоти (UI готов, логиката може да се добави)
- Hover ефекти върху карти
- Анимации при скролване
- Responsive дизайн
